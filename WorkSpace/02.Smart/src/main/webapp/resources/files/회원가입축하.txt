로그아웃상태에서 시작
메뉴 보여줌
메뉴 입력받고
1. 로그인 고르면
<true>일 때 정보수정, 게시판, 종료 메뉴 보여줌
  메뉴 입력받고
  1. 정보수정고르면 정보수정할건지 탈퇴할건지 골라
      수정할거면 1.수정메소드, 탈퇴할거면 2.탈퇴메소드
  2. 게시판 고르면 게시판 메소드
      게시판 메뉴 보여주고 원하는 기능 골라라
      -게시글 조회, 추가, 수정, 삭제, 종료
  3. 종료 누르면 종료
<false>일 때 로그인 재시도, 회원가입, 종료 메뉴 보여줌
  메뉴 입력받고
  1. 로그인 재시도 고르면 다시 로그인하고 맞으면 위에 정보 보여줌
